==========================================
!!! VIB Extensions (VIBEs)
==========================================

The Vector Integration Bus Extensions (VIBEs) are developed by Vector
and provided in binary form.
The VIBE Network Simulator allows more accurate simulations.
The :ref:`sec:util-registry` utility is also provided as a loadable extension in
shared library form.
The VIBE MDF4Tracing extension allows tracing VIB simulation messages in ASAM MDF4 file format.
