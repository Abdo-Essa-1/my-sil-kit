===================================================
VIB Tools and Demos
===================================================

The following documents describe the usage of the demo projects that are included with the Vector
Integration Bus project and what their expected output and or results are. All demo source code is
located in the GIT repository in the folder Demos.

.. admonition:: Note

   All paths on the following pages are relative to the top level of the GIT repository.
   Build artifacts are assumed to be located in a 'build' subdirectory.
